# 🖼️ Image Editing App with Streamlit & OpenCV

## Overview
This project is a browser-based Image Editing Application built using Python, Streamlit, and OpenCV. 
It allows users to upload an image, apply multiple filters in real time, preview the changes instantly, and download the edited image.

## Features
- Blur Filter
- Sharpness Adjustment
- Brightness Control
- Contrast Enhancement
- Grayscale Conversion
- Edge Detection
- Image Resizing (Width & Height)
- Side-by-Side Preview
- Download Edited Image as PNG

## Technologies Used
- Python
- Streamlit
- OpenCV
- NumPy
- Pillow
